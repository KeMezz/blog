## return types

## read-only property (immutability)

```typescript
type Age = number;
type Name = string;
type Player = {
  readonly name: Name;
  age?: Age;
};

const playerMaker = (name: string): Player => ({ name });

const nico = playerMaker("nico");

nico.age = 12;
nico.name = "hello"; // error! Cannot assign to 'name' because it is a read-only property.
```
